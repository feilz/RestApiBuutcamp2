This is a simple API to support a small blog application

It has all of the CRUD - methods available, and differentiates between different applications based on a key, given as a request parameter. You are free to choose whatever key you want. If you happen to select the same key as someone else, then you will share data. Below all available methods are listed, as well as sample data returned

You can add posts, and comments to the posts

Available methods:

Post methods
  • GET: /api/posts?key=''

    • Returns all posts created by your key

  • GET: /api/posts/{:id}?key=''

    • Returns a single post

  • POST: /api/posts?key=''

    • Create a new post, and assigns your key to it

  • PUT: /api/posts?key=''

    • Modifies an existing post

  • DELETE: /api/posts/{:id}?key=''

    • Deletes a post based on the id

Comment methods
  • GET: /api/posts/{:postId}/comments/

    Get all comments for a single post

  • GET: /api/posts/{:postId}/comments/{:commentId}

    Get a single comment for a post

  • POST: /api/posts/{:postId}/comments/

    Create a new comment for a post

  • PUT: /api/posts/{postId}/comments/

    Modify a comment for a post

  • DELETE: /api/posts/{postId}/comments/{:commentId}

    delete a comment from a post

When making calls to this api, you will get data in the following JSON format:

Post format when making a new Post:

{ "title": "SampleTitle", "post": "SamplePost, that is very cool" }

Comment format when making a new comment:

{ "text": "MyAwesomeComment" }
